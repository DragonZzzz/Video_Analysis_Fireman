# Video_Analysis

消防员操法打分项目，利用视频分类对消防员操法中出现的错误进行检测。

-----------------------------------------

###实现效果概述

- 消防员一开始甩水带→显示水带未压线 
  - (检测**压线**)
- 然后挂水带奔跑→显示未掉落
  - (检测**掉落**)
- 然后爬梯→显示第一步正确，未同节，未跨节
  - (检测**首步**、**同节**、**跨节**)
- 然后拉水带→显示未过分止线
  - ？？
- 操练结束

检测上述黑色部分的扣分项，输出最后分类结果。
